## 🎮 游戏流程

### 1. 连接阶段
```
用户打开页面 → 连接WebSocket → 显示连接状态
```

### 2. 登录阶段
```
点击登录 → 输入用户名 → 发送登录请求 → 显示登录状态
```

### 3. 房间阶段
```
创建/加入房间 → 显示房间信息 → 等待玩家加入
```

### 4. 游戏阶段
```
开始游戏 → 发牌 → 显示手牌 → 轮流出牌 → 计分 → 升级/下庄
```

### 5. 结束阶段
```
游戏结束 → 显示胜负 → 显示统计 → 返回大厅
```
