## 🎯 项目成果

### 1. 核心功能实现

#### 游戏规则
- ✅ **4人2队对抗**: 座位0&2为队A，1&3为队B
- ✅ **4副扑克牌**: 216张牌（含大小王）
- ✅ **主牌系统**: 大小王 > 主级牌 > 副级牌 > 主2 > 副2 > A...3
- ✅ **出牌规则**: 顺时针出牌，必须跟花色
- ✅ **分数计算**: 5/10/K各10分，总分480分
- ✅ **抠底规则**: 闲家抠底×2倍
- ✅ **升级规则**: 根据闲家得分升级/下庄

#### AI 玩家
- ✅ **三个难度级别**: 容易、中等、困难
- ✅ **智能出牌策略**: 根据游戏状态选择最优策略
- ✅ **记牌功能**: 困难模式记录已出牌
- ✅ **团队协作**: AI玩家考虑队友出牌

#### WebSocket 服务器

## 🚀 使用指南

### 1. 启动服务器

#### 方式1: 手动启动
```bash
cd /home/qmxz/cardgame-backend
php server.php
```

#### 方式2: 使用systemd服务
```bash
sudo systemctl start cardgame-ws
sudo systemctl status cardgame-ws
```

### 2. 访问前端界面

#### 测试客户端
```
http://localhost/test-client.html
```

#### 完整游戏界面
```
http://localhost/index.html
```

#### 增强版游戏界面
```
http://localhost/game.html
```

#### 简单演示
```
http://localhost/simple-demo.html
```

### 3. 游戏流程

1. **连接服务器**: 点击"连接服务器"按钮
2. **登录**: 输入用户名，点击"登录"
3. **创建/加入游戏**: 创建新游戏或加入现有游戏
4. **开始游戏**: 房主点击"开始游戏"
5. **出牌**: 选择卡牌，点击"出牌"
6. **结束回合**: 点击"结束回合"
